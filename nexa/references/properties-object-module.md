# MODULE MANAGER INFORMATION

## Package Name
- Description: Package identifier used for module publishing
- Type: `string`

## ModuleVersion
- Description: Module semantic version used for packaging and distribution
- Type: `string`
- Default: `1.0`

## ModuleDescription
- Description: Functional description used as metadata
- Type: `string`

## ModuleAuthor
- Description: Author metadata published with the module
- Type: `string`

## Module License URL
- Description: License reference URL for the module
- Type: `string`

## Module Project URL
- Description: Project homepage URL for the module
- Type: `string`

## Module Server URL
- Description: GeneXus Server URL associated to the module
- Type: `string`

## ModuleTags
- Description: Tags used to categorize and discover the module
- Type: `string`

## ModuleOwner
- Description: Owner metadata for governance and maintenance
- Type: `string`

## ModuleResources
- Description: File object resources packaged with the module
- Type: `string`
